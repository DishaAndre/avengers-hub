# Avengers Character Hub

A interactive web application built using React, inspired by a character showcase interface and redesigned with an Avengers theme.

---

## Tech Stack

* **Frontend:** React.js
* **Build Tool:** Vite
* **Language:** JavaScript
* **Styling:** CSS

---

## Project Structure

```
src/
 ├── components/
 │     ├── Navbar.jsx
 │     ├── Hero.jsx
 │     ├── Slider.jsx
 │
 ├── data/
 │     └── avengers.js
 │
 ├── styles/
 │     └── slider.css
 │     └── hero.cs  
 │     └── navbar.css
 ├── App.jsx
 ├── index.css
 ├── main.jsx
 ├── .gitignore
 ├── README.md
```

---

## Getting Started

Follow these steps to run the project locally:

```
git clone https://github.com/DishaAndre/avengers-hub.git
cd avengers-hub
npm install
npm run dev

```
